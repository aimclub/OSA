#### Аннотация

В данной статье исследуются методы контекстного понимания сленговых выражений и хэштегов в социальных сетях для повышения качества автоматической анонимизации и интерпретируемости коротких сообщений. В данной работе предлагается метод извлечения значений и синонимов сленговых выражений из небольшого числа коротких контекстов без дообучения моделей. Подход основан на использовании masked-token контекстных представлений, полученных с помощью компактных энкодермоделей, и последующей кластеризации этих представлений для выявления различных значений сленгового термина. Проведен анализ зависимости качества интерпретации от числа доступных контекстов. Экспериментальные результаты показывают, что устойчивые семантические кластеры могут 30-50 формироваться уже при наличии контекстов. что лелает предложенный метод перспективным практических задач ДЛЯ автоматической анонимизации, анализа текстов социальных сетей и повышения понимания коротких постов.

#### 1. Введение

Социальные сети являются одним из основных источников коротких пользовательских текстов. Подобные сообщения, как правило, содержат сленговые выражения, хэштеги, нестандартные формы слов и особые языковые конструкции, значение которых сильно зависит от контекста. При этом длина сообщений зачастую не превышает одного или двух предложений, что приводит к недостатку информации для однозначной интерпретации смысла.

Корректная интерпретация сленга является ключевой для целого ряда прикладных задач, включая автоматическую анонимизацию текстов, повышение их читаемости, анализ тональности и модерацию контента. Особую сложность представляет многозначность сленговых выражений, когда одно и то же слово может использоваться в нескольких различных значениях в зависимости от контекста.

Современные большие языковые модели демонстрируют высокую способность к генерации определений и перефразированию неформальной лексики. Однако их использование связано с существенными вычислительными затратами и зачастую не подходит для сценариев, где требуется приватность, воспроизводимость и работа на ограниченных ресурсах. Кроме того, такие модели могут неявно усреднять значения слов и при этом они не предоставляют явного механизма для анализа полисемии.

В данной работе исследуется возможность использования компактных предварительно обученных языковых моделей без дообучения для извлечения значений сленговых выражений из ограниченного числа коротких контекстов. Основная идея заключается в использовании masked-token метода для получения контекстных представлений удаленного сленгового слова и последующей кластеризации этих представлений с целью выявления различных семантических употреблений.

Основными вкладами работы являются:

- 1. предложение метода извлечения значений сленга без дообучения моделей;
- 2. эмпирический анализ зависимости качества интерпретации от числа контекстов.

# 2. Обзор существующих решений

# 2.1 Контекстная интерпретация значений слов

Ряд исследований показал, что контекстуализированные эмбеддинги, получаемые с помощью моделей семейства BERT, содержат информацию о

значении слова в конкретном контексте. Кластеризация таких представлений позволяет выявлять различные смыслы одного и того же слова без использования размеченных данных.

Тем не менее, большинство существующих работ ориентировано на формальные тексты и длинные контексты, а также предполагает наличие большого числа примеров для каждого слова. Проблема извлечения значений сленговых выражений из коротких и шумных текстов остается недостаточно изученной.

## *2.2 Masked language models и семантическое представление*

Masked language models обучаются предсказывать скрытые токены на основе окружающего контекста, что делает их естественным инструментом для восстановления семантического содержания удаленного слова. Скрытое представление в позиции [MASK] может рассматриваться как вектор ожидаемого значения слова в данном контексте.

Несмотря на это, masked-token представления редко используются для систематического анализа сленга и полисемии в условиях ограниченного числа контекстов.

# *2.3 Генерация определений и нормализация сленга*

Существующие методы обработки сленга в основном фокусируются на задаче нормализации, сопоставляя сленговые формы стандартным словам с использованием размеченных корпусов. Более новые подходы используют LLM для генерации определений и пояснений, однако такие методы требуют крупных моделей и не обеспечивают явного учета многозначности.

Таким образом, существует разрыв между мощными, но ресурсоемкими методами и легкими, интерпретируемыми решениями, пригодными для практического использования.

#### 3. Методология

### 3.1 Сбор и подготовка данных

На первом этапе из исходного корпуса коротких сообщений отбираются тексты, содержащие целевое сленговое выражение. Далее сленговый токен заменяется на специальный маркер [MASK], формируя анонимизированную версию текста.

Данный шаг позволяет одновременно решить задачу анонимизации и подготовить данные для masked-token метода.

### 3.2 Контекстные представления

С помощью предварительно обученной модели BERT извлекается скрытое представление в позиции [MASK]. Этот вектор интерпретируется как контекстно-зависимое семантическое представление удаленного сленгового слова.

Для оценки числа различных значений сленгового выражения применяется кластеризация эмбеддингов с автоматическим выбором числа кластеров. Перебирается диапазон значений k, для каждого из которых вычисляется silhouette score. Значение k, максимизирующее эту метрику, принимается в качестве оценки числа значений.

### 3.3 Кластеризация контекстов

После выбора оптимального числа кластеров выполняется кластеризация методом k-means. Каждый кластер интерпретируется как отдельное значение сленгового выражения, а центроид кластера используется как агрегированное представление этого значения.

# 3.4 Декодирование смысла кластеров

Для повышения интерпретируемости результатов используется два подхода.

Во-первых, центроиды кластеров сравниваются с эмбеддингами слов из словаря модели, что позволяет получить список ближайших по смыслу лемм.

Во-вторых, для каждого кластера отбираются наиболее репрезентативные контексты, близкие к центроиду. Эти примеры могут использоваться для последующей генерации кратких определений с помощью генеративных моделей.

### *3.5 Анализ зависимости качества от числа контекстов*

Для изучения устойчивости метода проводится серия экспериментов с увеличением числа контекстов N. Для каждого значения N фиксируются показатели качества кластеризации и стабильность числа выявленных значений.

# **4. Эксперименты**

Эксперименты проводятся на корпусе коротких сообщений из социальных сетей, содержащих распространенные английские сленговые слова. Используется компактная предварительно обученная модель BERTbase-uncased без дообучения. В качестве основных метрик рассматриваются silhouette score, устойчивость кластеров и интерпретируемость результатов.

## **5. Результаты**

Результаты показывают, что при очень малом числе контекстов (менее 10) семантическая структура выражена слабо. Однако при увеличении числа примеров до 50-70 формируются устойчивые и интерпретируемые кластеры.

Masked-token представления демонстрируют более высокое качество кластеризации по сравнению с sentence-level эмбеддингами, особенно для многозначных сленговых выражений.

Полученные результаты подтверждают, что masked-token методявляется эффективным инструментом для извлечения семантики сленга в условиях ограниченного контекста. Метод не требует дообучения, масштабируется на новые выражения и может быть интегрирован в практические системы анонимизации и анализа текста.

Основными ограничениями подхода являются чувствительность к шумным данным и зависимость от качества исходного корпуса.

### **6. Заключение**

В работе предложен метод извлечения значений и синонимов сленговых выражений из коротких текстов с использованием компактных языковых моделей без дообучения. Экспериментальные результаты показывают, что даже при ограниченном числе контекстов возможно выявление устойчивых семантических значений, что делает предложенный подход перспективным для широкого круга прикладных задач.
